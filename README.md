# Description

“EduTask”. It stands for Education Task. “EduTask” is an app built to connect students and teachers, making schoolwork organized and simple.

For students, you can now see all your assignments, projects, and school work in one place. You will never ever miss a due date again because one of “EduTask”  features is you can view the task's deadline. Making sure that you finished your task on time.

For teachers, You can now easily view your students' to-do lists. This feature allows you to monitor and provide extra help to a student when he/she needs it.
EduTask is where organization meets collaboration for a better school year.
